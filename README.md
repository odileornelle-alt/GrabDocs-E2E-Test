# GrabDocs E2E Testing

## Overview
This project contains an End-to-End (E2E) automated test script for GrabDocs using Selenium in Python.  
The script simulates real user actions on the platform.

## Features Tested
- Login
- File Upload
- Workspace Creation
- Messaging

## Tools Used
- Python
- Selenium WebDriver

## Test Script
- test_script.py

## How to Run
1. Install Selenium:
   pip install selenium

2. Download ChromeDriver and make sure it matches your browser version.

3. Update the script with your login credentials and file path.

4. Run the script:
   python test_script.py
